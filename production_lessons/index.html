Production track

Learn the practices that make code safe to ship and teams easier to work with.

These lessons cover code quality, Git workflow, GitHub collaboration, common Git problems with practical fixes, and deployment to Vercel from GitHub.

When to use this track

Start here when you want to move from writing code to maintaining it professionally.

Code quality

Learn how to organize frontend code so future changes feel safe and predictable.

Git confidence

Practice the commands you use every day so version control feels normal instead of scary.

Troubleshooting

Understand common errors and how to recover without losing work.
