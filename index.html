Practice-first frontend roadmap

Learn HTML, CSS, and JavaScript by building small, focused projects.

Every folder in this repo is a lesson with runnable code and a clear explanation of what the code is doing. Start at lesson 1 and work forward, then move into the production track for Git, deployment, and team workflow.

Start lesson 1 Open production track

How to study this repo

Read the notes, run the file, then change the code.

1. Read first

Each lesson folder has a README.md that explains the concepts and the code structure.

2. Run second

Open the lesson index.html in the browser and confirm you understand what the page should do.

3. Edit third

Change values, styles, and logic. Learning gets deeper when you test your own ideas.
