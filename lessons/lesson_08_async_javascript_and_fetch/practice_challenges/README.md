# Practice Challenges

## Challenge: Build an API-Powered Discovery Panel

Create a small interface that fetches data from an API and handles loading and failure states gracefully.

### Requirements

- Add a button that triggers a `fetch` request
- Show a loading message while the request is in progress
- Show the returned data in the page
- Handle errors with a visible fallback message or fallback content

### Stretch goals

- Add a second API card
- Disable buttons while loading
- Add a last-updated timestamp after each successful request
