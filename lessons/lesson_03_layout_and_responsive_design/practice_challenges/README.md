# Practice Challenges

## Challenge: Build a Responsive Learning Dashboard

Create a dashboard that looks good on desktop and still reads clearly on smaller screens.

### Requirements

- Add a hero section with summary cards
- Use CSS Grid for the main card layout
- Use Flexbox where one-dimensional layout is more appropriate
- Add a media query that changes the layout on smaller screens

### Stretch goals

- Add a sidebar that collapses below the main content on mobile
- Add progress cards with different sizes
- Test the page at multiple widths and adjust spacing
