# Practice Challenges

## Challenge: Build an Interactive Focus Tracker

Create a small app that updates the page when the user interacts with buttons.

### Requirements

- Add at least three buttons that change the UI
- Update text in the DOM based on user actions
- Add or remove a class to change the page appearance
- Create new activity items dynamically and insert them into a list

### Stretch goals

- Add a button that removes points
- Limit the activity log to the newest five actions
- Add a visual theme toggle

### Visual aid

- `visuals/challenge-guide.webp`: a storyboard-style UI reference that shows interactive states
