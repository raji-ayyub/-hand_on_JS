# Practice Challenges

## Challenge: Build a Study Plan Manager

Use arrays and objects to render a list of lessons and show different filtered views.

### Requirements

- Create an array of lesson objects
- Add buttons for "all", "completed", and "upcoming"
- Render the lessons into the page with JavaScript
- Show summary stats such as total lessons and total study time

### Stretch goals

- Add filtering by duration
- Add a difficulty level to each lesson object
- Show a count of unfinished lessons

### Visual aid

- `visuals/challenge-guide.webp`: a lesson-list reference
