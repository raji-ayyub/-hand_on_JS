# Practice Challenges

## Challenge: Build an Expanded User Directory

Extend the capstone idea into a more feature-rich mini app that looks and feels production-ready.

### Requirements

- Fetch user data and render profile cards
- Add search by name
- Add a company filter
- Show summary stats such as total users and visible users
- Handle both empty results and fallback data clearly

### Stretch goals

- Add sorting by city or name
- Add a detail view for a selected user
- Save the last search term in `localStorage`
