# Practice Challenges

## Challenge: Style a Mentor Profile Card

Turn plain markup into a polished profile card using spacing, color, borders, shadows, and hover states.

### Requirements

- Build a profile card with an avatar, role label, biography, skill tags, and a call-to-action button
- Use CSS variables for repeated colors and spacing
- Add hover feedback to the button
- Show clear use of padding, border radius, and box shadow

### Stretch goals

- Add a subtle card hover effect
- Create a secondary card variation with a different accent color
- Add a small status badge such as "Available for mentoring"
