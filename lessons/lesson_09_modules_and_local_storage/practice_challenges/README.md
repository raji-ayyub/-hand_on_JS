# Practice Challenges

## Challenge: Build a Saved Resource Board

Organize the code into modules and store data in the browser so the app survives refreshes.

### Requirements

- Split the logic into modules such as storage, UI, and app control
- Add a form for saving resource links
- Save entries in `localStorage`
- Render the saved resources back into the page after refresh

### Stretch goals

- Add editing for existing resources
- Add sorting by title or category
- Add a "clear all" action with confirmation
