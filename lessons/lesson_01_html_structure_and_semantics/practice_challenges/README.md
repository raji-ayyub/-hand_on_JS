# Practice Challenges

## Challenge: Build a Study Club Landing Page

Create a simple single-page layout for a frontend study club using semantic HTML.

### Requirements

- Use `header`, `nav`, `main`, `section`, `article`, `aside`, and `footer`
- Add a navigation menu with internal links
- Include a topics section, a meeting schedule section, and a signup form
- Connect every form input to a visible label

### Stretch goals

- Add an FAQ section
- Add a contact link in the footer
- Improve the heading structure so the page is easy to scan
