# Practice Challenges

## Challenge: Build a Workshop Signup Form

Create a form that validates user input and gives clear feedback before submission succeeds.

### Requirements

- Include name, email, track, and goal fields
- Prevent submission when the form is incomplete or invalid
- Show visible feedback messages for errors
- Render a confirmation summary after a successful submission

### Stretch goals

- Add a phone number field
- Add a checkbox for agreement
- Append multiple successful submissions instead of replacing the previous result

- `visuals/challenge-guide.webp`: a storyboard-style form validation reference
