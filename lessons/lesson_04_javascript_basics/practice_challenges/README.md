# Practice Challenges

## Challenge: Build a Score Summary Panel

Use JavaScript values, operators, and conditionals to update a study score panel in the browser.

### Requirements

- Store learner values with `const` and `let`
- Calculate totals or averages with operators
- Use template literals for the output
- Use at least one conditional to change the message based on the score

### Stretch goals

- Add a button for a second learner profile
- Display pass or fail status with different colors
- Show a progress percentage toward a target score
